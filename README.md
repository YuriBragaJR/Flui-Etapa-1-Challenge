# FLUi - Enterprise Challenge (Etapa 1)

Bem-vindo ao repositório do **FLUi**, uma solução de "Smart Charging" desenvolvida para a Etapa 1 do Enterprise Challenge. O aplicativo visa otimizar a experiência de motoristas de veículos elétricos na busca, planejamento e utilização de pontos de recarga.

## Sobre o Protótipo
Este projeto é um wireframe interativo desenvolvido para simular o fluxo principal do aplicativo FLUi. Ele apresenta uma interface fluida, moderna e focada na usabilidade (UX/UI), aplicando conceitos de design sustentável e tecnologia de mobilidade elétrica.

### Principais Funcionalidades
- **Splash Screen Animada:** Feedback visual imersivo simulando recarga elétrica (efeitos *neon flicker* e *electric-glow*).
- **Mapa Interativo:** Visualização de pontos de recarga próximos com indicação de status em tempo real (Livre, Ocupado, Manutenção).
- **Lista de Exploração:** Filtros e listagem dos postos disponíveis na região.
- **Ficha Técnica do Ponto:** Tela detalhada com endereço, custos, tipos de conectores (ex: CCS2, Type 2) e facilidades do local (Wi-Fi, café, banheiro).
- **Navegação Bottom Bar:** Acesso rápido e ergonômico às áreas principais do aplicativo (Mapa, Lista, Perfil).

## Identidade Visual e UX
- **Paleta de Cores:** Foco no tema escuro ("Noite de Carbono") para reduzir a fadiga visual, com elementos de destaque em "Verde Cinético" e "Azul Ionizado".
- **Tipografia:** Uso das fontes *Outfit* e *Inter* para garantir uma estética limpa, tecnológica e legível.
- **Ícones e Simbologia (Logo):** A marca une visualmente uma gota (fluidez) e uma folha (sustentabilidade), integrando um plugue de energia em seu espaço negativo.

## Acessibilidade
- **Alto Contraste Visual:** Cores de destaque vibrantes sobre fundo escuro para facilitar a leitura.
- **Feedback Duplo:** Comunicação de status por cores acompanhadas de rótulos de texto (ex: alerta vermelho acompanhado do texto "MANUTENÇÃO").
- **Alvos de Toque (Touch Targets):** Botões e cards dimensionados com padding amplo para prevenir toques acidentais, ideal para uso em movimento.
- **Iconografia Descritiva:** Uso de símbolos globalmente reconhecidos para reduzir a carga cognitiva.

## Tecnologias Utilizadas
- **HTML5** (Semântica e Estruturação)
- **CSS3 / Tailwind CSS (via CDN)** (Estilização, responsividade e animações personalizadas)
- **JavaScript (Vanilla)** (Lógica de navegação entre abas, renderização de dados simulados e interatividade)
- **FontAwesome 6.0** (Iconografia)
- **Google Maps Embed** (Visualização estática imersiva do mapa)

## Como Executar o Projeto

1. Clone este repositório em sua máquina local:
   ```bash
   git clone <url-do-repositorio>
   ```
2. Acesse a pasta do projeto.
3. Dê um duplo clique no arquivo `.html` principal do wireframe (ou arraste-o para o navegador).
4. O protótipo abrirá imediatamente no seu navegador padrão (Chrome, Firefox, Edge, Safari). Não é necessária a instalação de nenhuma dependência extra via NPM ou Node.js para visualizar o wireframe.
